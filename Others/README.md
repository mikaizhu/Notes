- https://zhuanlan.zhihu.com/p/31706200

论文页码从第三页开始：https://zhuanlan.zhihu.com/p/25923963

思路就是：

- 鼠标定位到第二页 布局 分隔符 分节符 连续
- 鼠标定位到第三页 双击页脚 取消链接上一页
- 插入页码 设置从第0页开始
