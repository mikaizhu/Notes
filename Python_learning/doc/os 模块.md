1. **系统相关内容**

```
os.name
os.environ
os.sep
os.pathsep
```

2. **文件和目录操作**

```
os.mkdir('filename')
os.rmdir('filename')
os.remove() # 删除文件
```

**获得目录**

```
os.getcwd()
os.chdir()
```

``` 
os.path.split() # 分隔文件名字和路径
os.path.isabd() # 判断路径是不是绝对路径
os.path.isfile() # 判断是不是文件
os.path.isdir() # 判断是不是路径
os.path.exists() # 判断文件是不是存在
os.path.getctime() # 获得文件的创建时间
os.path.getsize() # 获得文件字节
```

3. 操作系统

```
os.system('filename.py') # 执行某个py文件
```
